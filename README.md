# League of Legends Match Analzyer

A data pipeline that processes up to one year of a League of Legends player's ranked match history, providing a detailed breakdown of performance on a champion-by-champion basis.

## Features

- Fetch all ranked match IDs from the past year
- Retrieve full match data and metadata
- Parse and aggregate champion performance stats:
  - Win rate
  - KDA
  - CS per game
  - Gold, damage dealt, damage taken, vision score share, and kill participation
- Sorts champions by win rate and number of wins
- Outputs results to `output.json`

## Notes

- Handles early surrender games.
- Rate-limiting handled with `time.sleep(1)` between requests.
